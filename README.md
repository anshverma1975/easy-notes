# easy notes

A clean, minimal note-taking app with a pastel glassmorphism aesthetic. Organize notes into stacks, write freely, save instantly.

![easy notes](https://img.shields.io/badge/built%20with-html%20css%20js-CFE1FF?style=flat-square)

## features

- **Stacks** — group your notes into named collections
- **Notes** — create, edit, and delete notes within any stack
- **Auto-save friendly** — `Cmd/Ctrl + S` saves the current note
- **Deployable** — works on Vercel out of the box

## stack

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, no build step.

## running locally

Just open `index.html` in your browser. Or use a local server:

```bash
# python
python3 -m http.server 3000

# node
npx serve .
```

Then go to `http://localhost:3000`.
